A science-focused, more humane R interface to AWS.

## Installation

``` r
remotes::install_github("seankross/sixtyfour@main")
```

## Getting Started

```
Sys.setenv(
  AWS_ACCESS_KEY_ID = "",
  AWS_SECRET_ACCESS_KEY = "",
  AWS_REGION = "us-west-2"
)
```
